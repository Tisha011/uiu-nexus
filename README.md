# UIU NEXUS 🚀

A comprehensive Student Operating System designed for the Route 66 Hackathon. This platform integrates financial tracking, task management, and campus navigation.

## Features
- **Finance Tracker (Smart Wallet):** Track monthly allowances and daily expenses with automatic balance calculation.
- **Task Tracker:** Manage daily study goals and tasks with check-off functionality.
- **Campus Mini-Guide:** Quick access to campus building information and local services.

## Live Demo
You can access the live web application here: 
[https://uiu-nexus.vercel.app](https://uiu-nexus.vercel.app)

## How to explore the code
- **Main Logic:** Go to the `src` folder.
- **Components:** Check `App.jsx` for the primary dashboard logic.
- **Configuration:** Refer to `vite.config.js` and `tailwind.config.js` for build settings.

## Built With
- React.js
- Vite
- Tailwind CSS
